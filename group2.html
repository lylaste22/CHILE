Task Description

The objective of our work was to develop a classifier of different types of astronomical objects. To achieve this goal, we count with 25 features including celestial position, flux and ellipticity. Our main goal was to discriminate between stars, galaxies, asteroids and artifacts

For this purpose, we first distinguished between moving and non moving objects using DBSCAN. Then, we classified non-moving objects as stars, galaxies or artifacts. Finally, within our moving objects set, we searched for asteroids, using the Hough Transform and DBSCAN. This way, we found sources in different epochs which belonged to the same astronomical object.

Moving/non-moving object classifier

To do the classification between moving and non-moving objects, we ran the DBSCAN clustering algorithm on a dataset which included the detections on all 20 epochs. DBSCAN is a density-based clustering algorithm which puts in the same cluster two elements if they have a distance less than a specific threshold, and it doesn't need the number of clusters as a parameter. In this case, each cluster represented a non-moving astronomical object, while non clustered objects represented a moving object than can either be an asteroid or an artifact. We used 0.003 degrees (about 1.08 arseconds) as the distance threshold parameter for DBSCAN and only considered clusters with at least three samples.

Stars/Galaxied detection

Between all the features SExtractor gives, there is a CLASS_STAR feature that is a value between 0 and 1 which represents how confident is SExtractor to classify this detection as a star. We assumed that we could label as Star every object with a CLASS_STAR value greater than 0.9 and label as Galaxy all the objects with less than 0.1 in this feature. For each cluster, we considered the median of all its members.

After this, we used supervised learing to clasify the indetermined objects based on our labeled set and 24 features given by SExtractor. To test this method we splited our labeled set and created a training set from a random 50% of the data of the labeld set and leaving the rest to be part of the testing set. We got an accurracy of 95%.

To do the classification, we reduced the dimensionality of the data from 24 to 10 features using PCA, and then we classified it using Random Forest and Linear Discriminant Analisis. All the data which resulted with the same label with both of the classifiers where labeled either as a star or galaxy, while the rest was discarded and treated as an artifact.

Asteroid detection

Once we already had a dataset of moving-objects, we had to determine when sources of different epochs corresponded to the same astronomical object. To do that, we worked with the asumption that we could aproximate the trajectory of every asteroid to a straight line, and then, search for colinearity between sources. We considered all the lines between pairs of detections using a modified version of Hough Transformation in three dimensions, using the following ecuations

\theta = arctan\left(\frac{y_1 - y_2}{x_1 - x_2}\right)
x_0 = arctan\left(\frac{x_1 t_2+x_2 t_1}{t_1 - t_2}\right)
y_0 = arctan\left(\frac{y_1 t_2+t_2 t_1}{t_1 - t_2}\right)
\nu = arctan\left(\frac{y_1 - y_2}{x_1 - x_2}\right)

After doing this parametrization, we runned DBSCAN on top of a dataset of four dimensions in which each point represented a straight line which passes through two observation points. In our new space, high density spots would represent places where there are many straight lines coinciding, therefore we would find colinearity between several point in the original space.
