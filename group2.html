Task Description

The objective of our work was to develop a classifier of different types of stellar objects. We counted with 25 features including celestial postition, flux and ellipticity. Our main goal was to discriminate between stars, galaxies, asteroids and artifacts

For this purpuse, we first distinguished between moving and non moving objects using DBSCAN. Then, we classiffied non-moving objects as stars or artifacts. Finally within our moving objetcs set, we searched for asteroids, using Hough Transform and DBSCAN. This way, we found sources of different epochs which belonged to the same astronomical object.

Moving/non-moving object clasifier

To do the classification between moving and non-moving objects, we runned DBSCAN clustering algorith on top a dataset which included the detections of all 20 epochs. DBSCAN is a density-based clustering algorith which puts in the same cluster two elements if they have a distance less than a scpecific threshold, and it doesn't need the number of cluster as a parameter. In this case, each cluster represented a non-moving astronomical objects and non clustered objects represented moving object than can either be asteroids or artifacts. We used 0.003 degrees as the distance threshold parameter for DBSCAN and only considered clusters with at least three samples.
